
### Инициализация проекта

```bash
yarn create quasar
# или
npm init quasar
```

```bash
yarn global add @quasar/cli
# или
npm i -g @quasar/cli
```

```console
$ cd <project_folder_name>
# then run:

# if you have the global CLI:
$ quasar dev
# otherwise:
$ yarn quasar dev # or: npx quasar dev
```
