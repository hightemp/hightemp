# Словарь терминов

Коротко: общий словарь сложных терминов, которые встречаются в заметках `garden/`.

## Основное

- Словарь нужен для терминов, которые обычный читатель, скорее всего, не знает без специальной подготовки.
- Сюда стоит добавлять не бытовые слова, а понятия из математики, архитектуры, программирования, системного дизайна и соседних областей.
- Первое наполнение словаря сделано по заметке [The Mathematical Syntax of Architectures](../20-literature/202604271059-mathematical-syntax-of-architectures.md).

## Математика и архитектура

- **Синтаксис** - форма записи и структура объекта. В архитектурном контексте это ответ на вопрос: из каких элементов, связей и правил построена модель, если пока не обсуждать её смысл.
- **Семантика** - смысл того, что означает архитектура и её элементы в реальном мире.
- **Формализация** - перевод интуитивной идеи в точные правила, определения и математические конструкции, чтобы о ней можно было рассуждать без двусмысленности.
- **Математическая структура** - объект, описанный через набор сущностей и правил между ними.
- **Множество** - базовое математическое понятие: набор объектов, рассматриваемых как единое целое.
- **Отношение** - формальный способ описать, какие элементы связаны друг с другом.
- **Функция** - отображение, которое каждому допустимому входу ставит в соответствие один выход.
- **Под-архитектура** - выделенная часть архитектуры, которую можно рассматривать отдельно как самостоятельный структурный фрагмент.
- **View (представление)** - конкретный срез архитектуры для определённой цели.
- **Viewpoint** - правило или шаблон, по которому строят представление.
- **n-tier архитектура** - слоистая архитектура из `n` уровней, где связи разрешены только внутри слоя или между соседними слоями.
- **Гомоморфизм** - отображение между двумя структурами, которое сохраняет их важные структурные свойства. Если упростить: способ перевести одну архитектуру в другую так, чтобы не сломать её форму.
- **Морфизм** - более общее название для структурно корректного отображения между объектами.
- **Изоморфизм** - обратимый гомоморфизм. Если между двумя структурами есть изоморфизм, то по своей форме они эквивалентны.
- **Сюръективное отображение** - отображение, которое покрывает все элементы целевой структуры: у каждого элемента результата есть хотя бы один прообраз.
- **Категория** - математическая конструкция, где есть объекты и допустимые стрелки-морфизмы между ними, причём эти стрелки можно композиционно соединять.
- **Категория `Arch`** - категория, где объектами считаются архитектуры, а стрелками между ними - архитектурные гомоморфизмы.
- **Композиция** - последовательное применение преобразований: сначала одно, потом другое.
- **Теорема** - утверждение, которое не просто сформулировано, а доказано.
- **No-go theorem** - теорема о невозможности: показывает, что при выбранных предпосылках некоторую цель в принципе нельзя строго определить или получить.
- **Модульность** - свойство системы быть разбитой на относительно самостоятельные части с понятными границами и ответственностями.
- **Синтаксический уровень** - уровень рассуждения, где важна только форма структуры: какие есть элементы, связи и правила построения, но не их предметный смысл.
- **ADL (Architecture Description Language)** - язык описания архитектур, пригодный для точной записи, анализа и обработки архитектурных моделей.

## Как пополнять

- Добавлять только те термины, которые действительно требуют пояснения.
- Давать определения простым языком, без лишней академической тяжести.
- Если термин пришёл из конкретной заметки, добавлять источник в раздел `Источники`.
