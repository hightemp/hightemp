# Что такое Xposed

Источник: https://apptractor.ru/info/apps/xposed.html

Дата публикации: 2024-08-15

## Ключевые идеи

- Xposed Framework представлен как платформа для системной кастомизации Android без замены ROM или изменения базовой ОС.
- Модули Xposed работают поверх существующей системы: их можно включать, отключать или удалять, если изменения вызывают проблемы.
- Совместимость зависит от версии Android: для разных версий системы может требоваться своя версия фреймворка.
- Для установки обычно нужны root-права и Xposed Installer, через который пользователь управляет модулями.
- В технической части статья связывает Xposed с процессом Zygote: изменения внедряются так, чтобы новые приложения наследовали возможность перехвата поведения.
- Модули могут перехватывать Android API и системные вызовы, подменяя параметры, возвращаемые данные или поведение приложения.
- Примеры модулей: GravityBox для UI-твиков, Greenify для энергосбережения, XPrivacy для контроля приватных данных.

## Что забрать в permanent notes

- Главная идея Xposed - контроль поведения через перехват в runtime, а не через статическую модификацию APK или системного образа.
- С точки зрения архитектуры Android, Zygote делает такой подход мощным: одна точка внедрения влияет на множество последующих процессов.
