- pwnedOrNot - это инструмент OSINT (открытый источник информации) для поиска паролей скомпрометированных электронных почтовых аккаунтов. Он предоставляет возможность проверить, был ли аккаунт скомпрометирован в прошлом, используя API HaveIBeenPwned v3, а также искать пароль в общедоступных базах данных. Для использования инструмента требуется ключ API, который можно приобрести на сайте HIBP, ссылка на который приведена ниже. https://github.com/thewhiteh4t/pwnedOrNot

- Этот инструмент предназначен для поиска оригинального IP-адреса за веб-сайтами, защищенными CloudFlare; собранная информация может быть полезной для дальнейшего взлома.
