
### Репозитории

- **Kaku:** Высокопроизводительный терминал на базе WezTerm с открытым исходным кодом, оптимизированный для ускорения написания кода с помощью ИИ и предлагающий расширенную конфигурацию «из коробки», включая автоматическую смену тем, встроенные плагины для zsh и улучшенную отрисовку шрифтов. https://github.com/tw93/Kaku
- gptme — terminal-first open-source personal AI assistant и coding agent, который работает прямо в CLI, умеет редактировать файлы, выполнять команды, использовать инструменты и поддерживает локальные и удалённые модели через разные backends. https://github.com/ErikBjare/gptme
- Warp — open-source agentic development environment, выросшая из терминала: включает встроенного coding agent, поддерживает внешние CLI-агенты вроде Claude Code, Codex и Gemini CLI, открывает клиентский код на GitHub и позиционируется как AI-first terminal/workbench для разработки. https://github.com/warpdotdev/warp https://www.warp.dev/
