### Тестирование

1. **PHPUnit:** Это библиотека для написания и проведения unit-тестов на PHP. Фактически, является стандартом для покрытия кода тестами в огромном количестве PHP-проектов. Codeception использует PHPUnit внутри себя. [PHPUnit](https://phpunit.de/)
2. **Behat:** Это инструмент тестирования, основанный на концепции Behavior Driven Development (BDD). Он позволяет проверять ожидаемое поведение приложения на PHP от пользовательского уровня, используя простой и понятный естественный язык. [Behat](https://behat.org/)
3. **Kahlan:** Средство для unit и BDD тестирования на PHP со встроенной поддержкой стабов и моков, без необходимости дополнительных расширений. Это очень функциональная и гибкая тестовая рамка. [Kahlan](https://kahlan.github.io/docs/)
4. **Atoum:** Это простой, современный и интуитивно понятный инструмент для unit-тестирования на PHP. Он упрощает процедуру написания тестов и обеспечивает полное покрытие кода. [Atoum](https://atoum.org/)
5. **PHPSpec:** Это инструмент для тестирования поведения приложения, аналогичный Behat. Однако PHPSpec сосредоточен на тестировании кода на более низком уровне, ближе к коду, чем к пользовательскому интерфейсу. [PHPSpec](https://www.phpspec.net/)
6. **SimpleTest:** Это ещё один фреймворк для unit-тестирования в PHP. Он меньше и проще, чем PHPUnit, но все еще предоставляет все необходимые функции для успешного проведения unit-тестирования. [SimpleTest](https://simpletest.sourceforge.io/)
7. **Codeception:** представляет собой современную полнофункциональную систему для тестирования PHP-приложений. Инструмент поддерживает широкий спектр тестов, включая unit-тесты, функциональные тесты и приемочные тесты, и позволяет автоматизировать тестирование веб-приложений, API и даже бизнес-процессов. Codeception интегрируется с PHPUnit и Behat, обеспечивая удобный и гибкий подход к тестированию. [Codeception](https://codeception.com/)
8. https://github.com/Codeception/VisualCeption
9. architecture testing tool for PHP https://www.phpat.dev/
