### Скилы для AI-агентов

- CAD Skills — MIT open-source библиотека skills для CAD, robotics и hardware design agents: создаёт и точечно редактирует build123d/Python CAD parts и assemblies из текста или изображения, держит STEP/STP основным артефактом и экспортирует DXF, STL, 3MF, native GLB и topology sidecars; включает CAD Viewer, step.parts, URDF/SRDF/SDF robot/simulation descriptions, SendCutSend, G-code и Bambu Labs workflows. Ставится через `npx skills install earthtojake/text-to-cad` или native plugin marketplaces для Codex и Claude Code (`codex plugin add cad@text-to-cad`, `claude plugin install cad@text-to-cad`). https://www.cadskills.xyz/ https://github.com/earthtojake/text-to-cad
