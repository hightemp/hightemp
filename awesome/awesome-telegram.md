### Коллекции

### Репозитории MTproxy

- telemt/telemt — это высокопроизводительный MTProxy-сервер для Telegram, написанный на Rust с использованием асинхронного фреймворка Tokio. https://github.com/telemt/telemt
- 9seconds/mtg — это высокооптимизированный MTPROTO-прокси для Telegram, написанный на Go. https://github.com/9seconds/mtg

### MTProto серверы

- Teamgram Server — неофициальная open-source реализация MTProto-сервера на Go, совместимая с Telegram-клиентами и рассчитанная на self-hosted запуск; поддерживает MTProto 2.0, private chat, basic groups, contacts и web. https://github.com/teamgram/teamgram-server

### CLI и утилиты

- Chappe — open-source CLI surface для Telegram-каналов и AI-агентов: через TDLib синхронизирует историю постов и комментарии в локальную SQLite-базу, отдаёт JSON для Claude Code, Codex, OpenCode, OpenClaw и Hermes, умеет briefing, top posts/outliers, comments mining, ideas, draft/lint/preview и policy-gated publish с audit log. https://github.com/crimeacs/chappe
