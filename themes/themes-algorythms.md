**III. Основные алгоритмические парадигмы**

1. **"Разделяй и властвуй":**
    *   Принцип метода.
    *   Примеры: бинарный поиск, сортировка слиянием, быстрая сортировка.
    *   Рекурсия и рекуррентные соотношения.

2. **Динамическое программирование:**
    *   Принцип оптимальности подзадач.
    *   Перекрывающиеся подзадачи.
    *   Мемоизация (memoization) и табличный метод (tabulation).
    *   Примеры: числа Фибоначчи, задача о рюкзаке, наибольшая общая подпоследовательность, редакционное расстояние (расстояние Левенштейна).
    *   Восстановление решения.

3. **Жадные алгоритмы:**
    *   Принцип локальной оптимальности.
    *   Когда жадный алгоритм дает оптимальное решение.
    *   Примеры: алгоритм Прима, алгоритм Краскала, алгоритм Дейкстры, код Хаффмана.
    *   Эвристики

4. **Полный перебор (Brute-Force):**
    *   Метод грубой силы.
    *   Примеры: задача коммивояжера, задача о рюкзаке.
    *   Ограничения метода.
    *   Методы оптимизации перебора (отсечения, эвристики).
    *   Поиск с возвратом (Backtracking)

5. **Метод ветвей и границ:**
    *   Примеры задач: целочисленное линейное программирование, задача коммивояжера.
    *   Оценка верхней и нижней границы для отсечения ветвей.

**IV. Алгоритмы на графах**

1. **Обход графов:**
    *   Поиск в глубину (Depth-First Search, DFS).
    *   Поиск в ширину (Breadth-First Search, BFS).
    *   Применения: проверка связности, поиск циклов, топологическая сортировка.

2. **Кратчайшие пути:**
    *   Алгоритм Дейкстры (Dijkstra's algorithm).
    *   Алгоритм Беллмана-Форда (Bellman-Ford algorithm).
    *   Алгоритм Флойда-Уоршелла (Floyd-Warshall algorithm).
    *   Алгоритм A*.

3. **Минимальное остовное дерево:**
    *   Алгоритм Прима (Prim's algorithm).
    *   Алгоритм Краскала (Kruskal's algorithm).

4. **Связность:**
    *   Компоненты связности.
    *   Сильно связные компоненты.
    *   Мосты и точки сочленения.

5. **Потоки в сетях:**
    *   Задача о максимальном потоке.
    *   Алгоритм Форда-Фалкерсона (Ford-Fulkerson algorithm).
    *   Алгоритм Эдмондса-Карпа (Edmonds-Karp algorithm).
    *   Задача о минимальном разрезе.
    *   Теорема Форда-Фалкерсона.

6. **Паросочетания:**
    *   Задача о максимальном паросочетании.
    *   Алгоритм Куна для двудольных графов.
    *   Венгерский алгоритм.

7. **Эйлеровы и Гамильтоновы циклы:**
    *   Эйлеровы графы и циклы.
    *   Гамильтоновы графы и циклы.
    *   Задача коммивояжера.

8. **Раскраска графов:**
    *   Хроматическое число.
    *   Жадный алгоритм раскраски.
